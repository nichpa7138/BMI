เรียนรู้การสร้าง Web App คำนวณ BMI
ด้วยผู้ช่วย AI

บทเรียนนี้จะพานักศึกษาทำความเข้าใจตั้งแต่การวิเคราะห์โครงสร้างโปรแกรม (IPO) การเขียน Prompt สั่งงาน AI ไปจนถึงการนำเว็บไซต์ขึ้นออนไลน์ด้วย GitHub Pages

เริ่มบทเรียนกันเลย
Step 1

การวิเคราะห์ระบบ (System Analysis)

ก่อนเขียนโปรแกรม เราต้องวิเคราะห์โครงสร้าง IPO (Input - Process - Output)

I

1. Input

ข้อมูลที่ต้องรับเข้าจากผู้ใช้:

  • น้ำหนัก: กิโลกรัม (kg)
  • ส่วนสูง: เซนติเมตร (cm)
P

2. Process

ขั้นตอนประมวลผล:

1. แปลงส่วนสูงให้เป็น "เมตร" (หาร 100)

สูตร: BMI = น้ำหนัก / (ส่วนสูง)²

2. เปรียบเทียบเกณฑ์คนเอเชีย:

  • < 18.5 : น้ำหนักน้อย
  • 18.5 - 22.9 : ปกติ (สุขภาพดี)
  • 23.0 - 24.9 : ท้วม/ระดับ 1
  • 25.0 - 29.9 : อ้วน/ระดับ 2
  • >= 30.0 : อ้วนมาก/ระดับ 3
O

3. Output

สิ่งที่ระบบต้องแสดงผล:

  • ตัวเลข BMI: เช่น 24.2
  • ผลการประเมิน: ท้วม / โรคอ้วนระดับ 1
  • คำแนะนำ: ควรควบคุมอาหาร...
  • Extra: ปุ่มคำนวณใหม่ และรูปภาพ/Emoji ตามผลลัพธ์
Step 2

เทคนิคการเขียน Prompt สั่งงาน AI

นำข้อมูลจากการวิเคราะห์ระบบมารวบรวมเป็นคำสั่งที่ชัดเจน เพื่อให้ AI เขียนโค้ดให้

Prompt_For_AI.txt
คุณคือผู้เชี่ยวชาญการเขียน Web App ขอให้ช่วยเขียนโค้ดไฟล์ index.html สร้างระบบคำนวณค่า BMI (ดัชนีมวลกาย) สำหรับคนเอเชีย โดยใช้ Tailwind CSS ออกแบบให้สวยงาม ใช้งานง่าย โทนสีสดใส ข้อกำหนดระบบ (IPO): 1. Input: รับค่าน้ำหนัก (กิโลกรัม) และ ส่วนสูง (เซนติเมตร) 2. Process: แปลงส่วนสูงเป็นเมตร แล้วคำนวณ BMI = น้ำหนัก / (ส่วนสูง)² เกณฑ์เปรียบเทียบ: - < 18.5 : น้ำหนักน้อย - 18.5 - 22.9 : ปกติ (สุขภาพดี) - 23.0 - 24.9 : ท้วม / โรคอ้วนระดับ 1 - 25.0 - 29.9 : อ้วน / โรคอ้วนระดับ 2 - >= 30.0 : อ้วนมาก / โรคอ้วนระดับ 3 3. Output: แสดงตัวเลข BMI, ผลการประเมิน, คำแนะนำที่เหมาะสม, และแสดงภาพ Emoji ให้เหมาะสมกับผลลัพธ์ 4. เพิ่มเติม: มีปุ่มให้กด "คำนวณใหม่" ด้วย

Step 4

วิธีนำเว็บไซต์ขึ้นออนไลน์ด้วย GitHub Pages

เรียนรู้วิธีแปลงไฟล์ index.html ที่สร้างขึ้น ให้กลายเป็นเว็บไซต์จริงที่ทุกคนสามารถเข้าถึงได้ฟรีผ่านอินเทอร์เน็ต

1

สมัครสมาชิก GitHub

เข้าไปที่ github.com และทำการ Sign Up ให้เรียบร้อย ยืนยันอีเมลเพื่อเปิดใช้งานบัญชี

2

สร้าง Repository ใหม่

กดปุ่ม "New" เพื่อสร้าง Repository ตั้งชื่อโปรเจกต์ เช่น bmi-calculator และตั้งค่าเป็น Public

3

อัพโหลดไฟล์ HTML

คลิก "uploading an existing file" ลากไฟล์ index.html ของเรามาวาง แล้วกด Commit changes

4

เปิดใช้งาน Pages

ไปที่แท็บ Settings > Pages ตรง Source เลือก "main" branch แล้วกด Save รอ 1-2 นาทีจะได้ลิงก์เว็บไซต์ใช้งานได้เลย!
